03 · Decisions baked into v1 — rule on these
F1 · Per-row Auto cells + global Reset. Each grid leads with a dashed "Auto" cell, so you can override color while leaving shape hashed (the data model already supports independent absence). "Reset to auto" clears both. Alternative: a single Reset only, no per-row Auto.
F2 · Modal dialog, not anchored popover. One surface serves both entry points and survives small screens. Alternative: popover anchored to the hover card, modal only from the writing page.
F3 · Names appear in the preview caption only ("the wellspring"), never as grid labels — keeps the grid quiet, and display names ship only after the 06–16 roster reconcile (ruling 49). Shape cells get the name as a hover tooltip.

Mock caveats: silhouettes and inner designs are approximations drawn for this layout; the live picker reuses the in-repo verbatim table and the constellation's own renderer (halo · body · ring · shine · inner), so the preview equals the shipped mark byte-for-byte. Save persists sub.markShape / sub.markColor; Cancel and esc discard. Swatches always display the colorful identity hues; the global palette toggle mutes everything downstream.

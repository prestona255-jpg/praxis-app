  spread, via the `gatherable` arg to `renderNotebookEntry`) adds/removes it from a persisted
  gathered set; gathered cards get a `.notebook-entry-gathered` outline. The set persists across

  {kind:'entry', refId, quote})` per gathered note — the same path "Send to sub-theory" uses. On
  success, clears the gather and routes to `#subtheory/<id>`. **Zero new data path** (the locked
  model's REUSE requirement).
